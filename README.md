##haskell与javascript
作为前端开发语言，javascript越来越强悍，以至于nodejs可以运行在服务端，而目前win8和linux都可以用javascript写app。javascript也可以跨平台写ios和android等app。
haskell的函数式特性可以巧妙地节省开发时间，也就是说可以用haskell来写代码并编译成javascript。这样就十分高效了。
