##haskell
其实我原本不知道haskell，但我在三四月份时用周末时间给一个高中生辅导pascal时，发现他正在学scheme，并且是MIT的课程。虽然我以前也听说过函数式，但从未接触过。而pascal这种十分严谨的语言确实与函数式有着本质的不同。

scheme是lisp的一种方言。有很多相关的具体实现。它的语法十分古怪，括号多的令人眼花缭乱。正因为如此，我才选择了看起来不会让我晕头转向的haskell。
